### Environment

- Xperia 1 V
- Android 14
- [KernelSU](https://github.com/tiann/KernelSU/releases) lastest
- [Quickswitch](https://github.com/skittles9823/QuickSwitch/releases) V[4.0.2](https://t.me/QuickstepSwitcherReleases)
- [Lawnchair](https://github.com/LawnchairLauncher/lawnchair/releases) latest Nightly

### Quickstart

- Install Lawnchair as default launcher.
- Use KernelSU to flash Quickswitch.
- Run `/data/adb/modules/quickswitch/quickswitch --ch=app.lawnchair.debug` with root permission.
- Restart.

### FQA

- Lawnchair beta 2 has overlay stuck issue, so must use nightly.
- Quickswitch V4.0.2 is only available in Telegram groups.
